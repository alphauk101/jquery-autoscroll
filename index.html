AutoScroll

About

Autoscroll can do really cool things on your site with really little effort from programmer. This create a kind of interface that makes user don't need to do scrolling all time, he only moves the mouse through the surface and it will move smooth automatically.

In truly, I don't need to write a longer text like this only to explain the advantages of using this library, but I need a big text to use as demo in this main page, so you can scroll this text using my library.

Be happy with this, it works on most browsers.

Have fun!

Requirements

AutoScroll requires jQuery 1.2+

Installation

Just include autoscroll.min.js (or other version, all end at same) into your site head tag.
